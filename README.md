# Wave-and-Burgers-Equation

Solution of the wave equation and the burgers equation  with numerical schemes such as FTBS, and Lax,Lax-Wendroff. 
A jupyter file is provided along with a pdf with a discussion of processes and results.

*A project in the context of the course "Computational Aerodynamics" (spring semester 2025-2026 @University of Patras, Mech eng and aeronautics)*
